# DalamudPlugins

Dalamud custom plugin repository for all of my FFXIV plugins.

## Install

Add the following URL to *Dalamud Settings → Experimental → Custom Plugin Repositories*:

```
https://raw.githubusercontent.com/AlessHub/DalamudPlugins/main/repo.json
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [Action Wheel](https://github.com/AlessHub/ActionWheel) | A customisable radial wheel for quickly using emotes and macros |

## Adding a new plugin

1. Add an entry to `plugins.json` with the plugin's metadata and GitHub repo
2. Push — the `repo.json` will be rebuilt automatically on the next release dispatch
